# Ejemplo Email Sender/Receiver

Este proyecto Python permite enviar y/o recibir correos a través de un servidor SMTP y IMAP, respectivamente.

## Instalación

```bash
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
```
